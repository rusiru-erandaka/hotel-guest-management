# PocketBase Backend

## How to Run
1. Open a terminal in the `server/` folder.
2. Run the following command:
   ```powershell
   .\pocketbase.exe serve
   ```

3. PocketBase will start at http://127.0.0.1:8090
4. Admin UI is available at http://127.0.0.1:8090/_/
